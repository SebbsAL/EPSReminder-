🤖 Bienvenido a EPSReminder!

¡Olvídate de perder citas médicas o eventos importantes! EPSReminder! es tu asistente personal diseñado para gestionar recordatorios de salud y tareas personales directamente en tu Discord.

✨ ¿Cómo funciona?
Hemos simplificado todo para que no tengas que configurar nada. Tu experiencia comienza en nuestra plataforma web:

Visita nuestro sitio oficial: https://eps-reminder.vercel.app/

Únete a la comunidad: Desde la página principal, haz clic en el botón "¡Añadir a Discord!". Esto te llevará directamente a nuestro servidor donde el bot está listo para asistirte.

Agenda tus recordatorios: Una vez en el servidor, utiliza el formulario integrado en la web para registrar tus citas médicas o recordatorios de EPS en segundos.

🚀 ¿Por qué usar EPSReminder?
Sin instalaciones: Todo se gestiona desde el navegador y Discord.

Gestión Inteligente: Notificaciones automáticas directamente en tus canales o mensajes directos.

Diseño Amigable: Una interfaz intuitiva pensada para que agendar una cita no sea un dolor de cabeza.

❓ Preguntas Frecuentes
¿Es gratuito?
Sí, el servicio es totalmente gratuito para todos los usuarios.

¿Necesito instalar algún software adicional?
¡No! Solo necesitas tener una cuenta de Discord y acceso a nuestro sitio web.

¿Mis datos están seguros?
La información que ingresas es procesada exclusivamente para enviarte los recordatorios de manera eficiente y segura.
